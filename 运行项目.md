bin文件夹下的train.py是启动项目的文件
里面会去调用configs下的yaml文件
configs文件下的说明，training是训练场景的核心配置（重中之重），prediction是推理（图像修复）场景的配置
data_gen是掩码生成（数据增强）的配置，根目录eval*.yaml是训练后评估 / 测试的配置

我们以training下的big_lama.yaml文件为例
在他最下面的defaults字段下
defaults:
  - location: docker  # 环境配置（docker/本地）
  - data: abl-04-256-mh-dist  # 数据集配置的核心引用（重点！）
  - discriminator: pix2pixhd_nlayer  # 判别器配置
  - optimizers: default_optimizers  # 优化器配置
  # ... 其他模块引用

  data: abl-04-256-mh-dist表示big-lama.yaml会自动加载 configs/data/abl-04-256-mh-dist.yaml（数据集路径、图像尺寸等都写在这个文件里）。
  你去该文件里面看，就可以看到可以去修改数据集的路径
  同样的前面的名称比如discriminator，就去training下的discriminator文件夹下找。
